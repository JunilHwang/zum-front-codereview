## Front-End

```
# 패키지 설치
yarn
npm install

# 개발모드(핫리로드)로 실행
yarn dev
npm run dev

# 프로덕션으로 배포 하기 (경로 dist 디렉토리)
yarn build
npm run build
```

#### 실행 URL

> http://localhost:80

## Back-End

```
# 패키지 설치
yarn
npm install

# 서버 실행
yarn start
npm run start
```

#### 실행 URL

> http://localhost:3000
